# Network Scanner using Nmap

## Description
Python-based network scanner to identify open ports and running services on a target IP or domain using Nmap.

---

## Features
- Scan any IP address or domain
- Detect open port
- Identify services running on each port
- Fast scanning using SYN scan technique
- Simple command-line interface

---

## Technologies used
- Python
- Nmap
- python-nmap library

## Installation

### 1. Install Nmap
Download and install Nmap from official website.

### 2. Install Python library
```bash
pip install python-nmap
